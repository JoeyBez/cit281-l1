# Lab 1

Using basic functions in JavaScript, as well as getting familiar with the command line.
```ruby
function square(num) {
    return num * num;
}

console.log('Square operations:')
for(let i = 2; i <= 10; i += 2) {
    console.log(`Square of ${i} is ${square(i)}`);
}
```
